# ICalendar

## User Possibilities


- [ ] User can create event
- [ ] User can edit event
- [ ] User can delete event
- [ ] User can create a schedule
- [ ] User can see by priority
- [ ] Change theme (dark/light)
- [ ] User can drag events between dates
- [ ] User can set reminder for a event
